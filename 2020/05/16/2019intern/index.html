2019 Fall + 2020 Spring Software Engineer Intern Timeline 转专业找实习记录 + 面经

Posted on 2020-05-16 | In Programming , Interview

纯记录，时间顺序

Riot Games - Software Engineer Intern

2019.9.19 网申 当天收oa->9.25 完成oa->9.26 电面邀请-> 10.11 hr面 ->10.16 挂了。

没啥好说。

Goldman Sachs - Analyst Intern

2019.9.16 网申 -> 9.17 收到OA 隔天完成-> 9.11 收到HireVue -> 9.14 完成HireVue -> 10.9 Superday邀请（原定16号，midterm所以推迟到25号） -> 10.25 Superday -> 12.04 电话拒信

第一个正经面试。高盛一反常态的拖延结果，感觉可能被放备胎池了，但的确也面的不好，开学之后没刷题手生了，两道medium题都没做出来。在新泽西高盛的楼面的，安排的酒店挺舒服:O

两轮45min面试，基本形式是两个面试官bq拷问，再问两道题，在纸上大概写一写代码就行。第一轮问了两道easy题+一道follow up medium tree题，最后那道又没做出来，害。第二轮问了LRU Cache+Shuffle an Array，第二题当时没做过，没写出来。总之车祸面试。

Airbnb - Software Engineer Intern

九月底学姐内推->10.24 收到OA -> 10.28 Hello From Airbnb -> 11.7 两轮面试 -> 12.5 说坑满了（呵呵）

Airbnb基本只出题库题，并且难度都很大，这次准备的还是很认真的。考了file system和bank system，第二题没做过做的比较坎坷，不过还是写出来了。最后因为没坑挂了，当时失望归失望，今年疫情取消了offer，现在想想被挂倒不是坏事（人生啊，尽是没想到.jpg）

Databricks - Software Engineer Intern

2020.1.6 网申->1.7收到next steps->1.18 karat技术面->1.21 安排hr面->1.23 hr intro call 当天安排下一步->1.30 第二轮技术面->1.31 拒信

二面没做到面经题，面试官看之前的实习经历给我现场编了一题出来，沟通也差了点，最后磕磕绊绊做出来了，但是估计也是没达到面试官内心标准的，新学期第一个面试，anyway move on。夸赞hr效率很高。

UiPath - Software Engineer Intern

Software Engineer Intern1.15 网申->1.21 说安排phone interview ->1.24 hr面 说安排二月初面试->2.1拒信

没啥好说。。

TuSimple - Software Engineer Intern

1.16 网申->1.21 oa->1.28 完成oa 当天安排下一轮->2.11 一面->2.13 拒信

图森的题也是有名的难，而且不像airbnb有题库，当时其实在准备其他几家公司的面试，所以基本没准备，拒了也不意外。面试的国人小哥人特别好，因为我一点思路都没有。。完全靠引导解题，试图想起来题目，但是失忆了，一道解法挺巧的较难medium或较易hard题。

Snowflake - Software Engineer Intern

2.5 cmu cf 投->2.6 校园面->2.14 催出结果->2.18 二面->2.20拒信

校园面很顺利，问了一道很简单的用hashmap的题，follow up是二维变三维，还有一道tree题，follow up和Populating Next Right Pointers in Each Node差不多，在白板写题，最后一个follow up思路有点小问题，国人小哥一直在引导，最后一起做出来的。二面就很糟糕了，问了一道不难的非面经题，然后基本功拷问，炸了。

Schlumberger - Software Engineer Intern

2.5 cmu cf 投->2.7 校园面->2.9 oa ->3.30 拒信

校园面就是bq胡扯，聊的很开心，oa也万年不变，但最后被拒了，不过他们的summer intern offer也因为疫情revoke了，害。

Fanatics - Technology Intern

1.13 网申->2.4 技术面->2.6 安排下一轮->2.10 manager面->2.13 Offer

一家面试体验非常棒的公司，技术面面了 Decode Ways。base在三番，hr和面试的国人姐姐都很和善，实习薪资和补贴也不错的，看glassdoor感觉是氛围很棒的小公司，不过这个岗位做的是Data Engineer Intern，再加上也有收到其他offer所以拒了。

Uber ATG - Software Engineer Intern

1.28 handshake网申->2.3 next steps ->2.10 一面 当天安排下一轮->2.14 二面->2.19 manager面 当天Offer

Uber一路都面的很顺，一面就是一道简单的用binary search的题，follow up问了多线程还是分布式之类的概念题，什么多台机器处理 ip address之类的，一通胡扯竟然过了，二面题目忘了，，好像分别用iterative和recursive做了同一题。hr效率也很高。

Facebook - Software Engineer Intern

1.15 内推 ->2.6 安排一面->2.13 一面->2.24 二面->2.25 offer

两轮都是45min内两道面经题。一轮是两道变形的面经 task scheduler 和find anagram in string，二轮又忘了是啥，都是中高频面经就对了。fb面试不用跑代码，所以不需要面对一些意外bug，这两轮也都蛮顺利的。

接了fb之后，陆续收到一些面试（Oracle、NetApp、Nvidia、Cisco这些春季都在继续招），但找了半年实习实在是累了所以就没面，不容易。所以亚麻为什么简历拒我:(
